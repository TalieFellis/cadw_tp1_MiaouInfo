# cadw_tp1_MiaouInfo
MiaouInfo est une application Laravel qui présente des faits amusants sur les chats. L'utilisateur peut consulter un fait aléatoire sur la page d'accueil, voir la liste complète des faits, ajouter de nouveaux faits, et modifier ou supprimer des faits existants. Ce projet a été réalisé dans le cadre d'un travail scolaire.
